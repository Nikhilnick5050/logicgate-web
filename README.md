## üöÄ Overview

**Logic Gates Hub** is an interactive web application designed to help students understand the fundamental concepts of digital electronics and logic gates. The platform features detailed explanations, interactive truth tables, visual representations, and real-time simulations of various logic gates.

## ‚ú® Features

### üéØ **Interactive Learning**
- **Real-time Gate Simulation**: Toggle inputs and observe outputs dynamically
- **Visual Gate Representations**: Clear diagrams for each logic gate type
- **Truth Tables**: Interactive tables showing all possible input-output combinations
- **Boolean Expressions**: Mathematical representations of gate operations

### üîß **Logic Gates Covered**
- **AND Gate** - Outputs true only when all inputs are true
- **OR Gate** - Outputs true when at least one input is true
- **NOT Gate** - Outputs the inverse of the input (inverter)
- **NAND Gate** - Outputs false only when all inputs are true
- **NOR Gate** - Outputs true only when all inputs are false
- **XOR Gate** - Outputs true when inputs are different
